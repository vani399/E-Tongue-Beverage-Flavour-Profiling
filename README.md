# E-Tongue for Beverage Flavor Profiling and Quality Analysis Using Machine Learning

## Project Overview

The E-Tongue (Electronic Tongue) system is an AI-powered beverage quality assessment platform designed to mimic the human sense of taste using multiple sensors and machine learning.

The system collects real-time data from pH, conductivity, gas and temperature sensors and uses a Decision Tree Machine Learning model to classify beverages as Consumable or Non-Consumable.

## Features

* Real-time beverage quality monitoring
* pH level analysis
* Conductivity-based sweetness detection
* Gas concentration monitoring
* Temperature monitoring
* Machine Learning classification
* OLED display integration
* ThingSpeak cloud monitoring
* Consumable / Non-Consumable prediction

## Hardware Components

* Raspberry Pi 4
* Arduino UNO
* pH Sensor
* Conductivity Sensor
* Gas Sensor
* DHT11 Temperature Sensor
* OLED Display

## Software Used

* Python
* Google Colab
* ThingSpeak
* Thonny IDE
* Decision Tree Classifier

## Machine Learning

The system uses a Decision Tree model trained on:

* pH Values
* Conductivity Values
* Temperature
* Gas Concentration

Output:

* Consumable Beverage
* Non-Consumable Beverage

## Applications

* Beverage Quality Analysis
* Food Safety Monitoring
* Smart Food Industry
* Automated Taste Profiling
